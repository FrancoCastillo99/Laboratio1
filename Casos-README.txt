MUTANTES 
El ingreso por teclado son todas las letras seguidas, la separación que se ve es para que se nota donde se encuentra la secuencia, esto es solo para el README

1° Caso:
Ingrese la 1° secuancia de dna: A GGGG T
Ingrese la 2° secuancia de dna: C A CTTC
Ingrese la 3° secuancia de dna: GC A CCT
Ingrese la 4° secuancia de dna: TTC A GG
Ingrese la 5° secuancia de dna: ACTGTC
Ingrese la 6° secuancia de dna: CCTGGG

Es mutante

Ingrese la 5° secuancia de dna: CCGTTA
Ingrese la 6° secuancia de dna: ACGTGC
